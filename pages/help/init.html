Oldal aktivizálása

Githubbal készítjük a személyes oldalt. Ezen az oldalon tudod megmutatni magad vagy egy projektet.
  1. Menj a Githubra.
  2. Nyiss 1 új repositoryt a + (jobb felül) ikonnal.
  3. Add meg a nevet. (A repositorynak kötött a neve: [felhasználónév.github.io])
  4. Description (Leírás)
  5. Publikus vagy privát?
  6. Inicializáljuk a readme file-t.
  7. Majd Create repository KATT. :)
  8. Ha létrejött a repository le kell klónozni a saját gépre -> Katt [Clone or download] gombra. Majd kimásoljuk a klón URL-t.
  9. Nyissunk egy parancssort.
  10. Lépjünk ki ebből a mappából. (cd..)
  11. Ide klónozzuk a github demónkat (git clone [a kimásolt link])
  12. Belépek az új mappába (cd bodai81.github.io)
  13. Ebben a mappában nyitok 1 új Visual studio Code-t. (code .)
  14. A megjelenő új ablakban tudom szerkeszteni a projektet.
  15. Hogy gyorsabb legyen a dolog átmásolom az eredeti könyvtárból a dolgokat ide. (Fájlkezelőben mindent kijelöl ->átmásol az új projektes mappába.)
